Perguntas frequentes

Qual a diferença para o PDFWords?

O PDFWords extrai automaticamente todas as palavras dos PDFs (abordagem aberta, exploratória). O PDFTerms faz o contrário: você controla o vocabulário, listando exactamente os termos de interesse. É a abordagem dirigida, ideal para revisões sistemáticas e para testar conceitos pré-definidos.

Como agrupar sinônimos?

Use o sinal de igual. Exemplo: CO2 = carbon dioxide; dióxido de carbono. Todas as variantes são contadas como um único termo, com o rótulo à esquerda do "=". Separe as variantes por ponto-e-vírgula ou vírgula.

Como usar curinga e termos compostos?

O * captura variações: emission* casa emission, emissions, emitting. Termos compostos funcionam directamente: shore power procura a expressão inteira. A busca ignora maiúsculas e acentos.

O que é contagem binária por documento?

No escopo "Documento", dois termos co-ocorrem se aparecem no mesmo PDF, contando uma vez por documento. Isso evita que um PDF longo, que repete um termo muitas vezes, domine a rede. É o padrão recomendado em bibliometria. No escopo "Frase", a co-ocorrência é contada a cada frase em que os dois termos aparecem juntos.

Como as categorias aparecem no VOSviewer?

Cada categoria entre [ ] vira um cluster no JSON, e o VOSviewer atribui uma cor distinta a cada cluster automaticamente. Termos sem categoria ficam num grupo neutro.

Alguns termos não foram encontrados — e agora?

O resultado lista os termos com zero ocorrências, para controlo de qualidade. Verifique grafia, acentos, tente um curinga (ex.: govern*) ou adicione sinónimos. PDFs digitalizados (só imagem) precisam de OCR antes.
